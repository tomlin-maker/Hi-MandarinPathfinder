# Hi-MandarinPathfinder
Learning Path Filtering Tool to support Hi-Mandarin Instructors to facilitate Lead-in Sessions
系統簡介
Hi-Mandarin Lead-In Pathfinder 是一個由 Hi-Mandarin 設計開發提供 Hi-Mandarin Instructor (TA) 在 Lead-in Session 使用的 PC / MAC 離線 HTML工具及本線上版本的工具，協助 Learners 和教學顧問 (TA) 快速找到適合的學習路徑。透過選擇學習意圖(intent)，系統會自動產生包含預習影片、1-1 Tutor Lesson 和 TA 練習的完整學習建議。[v1.4後版本發佈在線使用 (提供多種設備支援)，但我們仍鼓勵您下載離線版，避免在線版失效]
主要功能
#意圖選擇：標籤搜尋
#學習路徑規劃：自動產生5步驟學習建議
#教材清單：顯示Pre-watch模組和Tutor Lesson列表
#複製路徑：一鍵複製簡版學習計畫到剪貼簿、或手動剪貼完整版學習計畫
#列印/匯出：產生適合列印的PDF格式
